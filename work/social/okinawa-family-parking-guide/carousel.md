```
標題　　停車場的機器都是日文

　　　　那幾秒特別焦慮。

　　　　但其實沖繩的投幣停車場
　　　　主要就這三種。
```

版面：上方 600px 照片橫幅（**重用文章情境圖** `parking-family-getout.jpg`，媽媽扶孩子下車），下緣用米白漸層淡出銜接文字，左下角標「情境示意」。

**原本規劃在此放一張精算機 AI 照片，已取消。** 生圖時鍵盤數字必定可讀（違反無可讀文字規則），且模型生不出日本精算機的樣子。改用文章既有的情境照更貼題，也零成本。

## P4｜型式一：鎖板式

```
標籤　　1 / 3　最常見
標題　　鎖板式（フラップ／ロック板式）

圖示　　上方 560px 照片橫幅：鎖板卡住車輪的特寫

步驟　　① 停進車格，地面鎖板自動升起卡住車輪
　　　　② 離開前到精算機，輸入車格號碼
　　　　③ 付款後鎖板才會降下
```

版面：上方照片橫幅用 `r1-flap-plate-v2.jpg`（本次生成，唯一通過驗收的一張），標「情境示意」。下方步驟用數字圓標 `#9A6B43`，一行一步。

## P5｜型式二：柵欄式

```
標籤　　2 / 3
標題　　柵欄式（ゲート式）

圖示　　入口柵欄 + 取票機的簡化插畫

步驟　　① 進場抽一張停車券，柵欄升起
　　　　② 離場把停車券放進精算機
　　　　③ 付款後柵欄升起才出得去

提醒　　停車券不能弄丟，弄丟通常要按規定補繳。
　　　　帶孩子上下車時把票收好。
```

版面：提醒用 `#B4552F` 文字 + 淺色底框。

## P6｜型式三：車牌辨識式

```
標籤　　3 / 3
標題　　車牌辨識式（ロックレス式）

圖示　　攝影機 + 車輛的簡化插畫

步驟　　① 沒有鎖板也沒有柵欄，靠攝影機辨識車牌
　　　　② 離場前到精算機，輸入車格或車牌
　　　　③ 付款完成

提醒　　最不會卡到車，但也因為沒有實體阻擋，
　　　　最容易忘記付款這一步。
```
