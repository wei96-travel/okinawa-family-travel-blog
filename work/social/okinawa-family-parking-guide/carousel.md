## P5｜型式二：柵欄式

```
標籤　　2 / 3
標題　　柵欄式（ゲート式）

圖示　　入口柵欄 + 取票機的簡化插畫

步驟　　① 進場抽一張停車券，柵欄升起
　　　　② 離場把停車券放進精算機
　　　　③ 付款後柵欄升起才出得去

提醒　　停車券不能弄丟，弄丟通常要按規定補繳。
　　　　帶孩子上下車時把票收好。
```

版面：提醒用 `#B4552F` 文字 + 淺色底框。

## P6｜型式三：車牌辨識式

```
標籤　　3 / 3
標題　　車牌辨識式（ロックレス式）

圖示　　攝影機 + 車輛的簡化插畫

步驟　　① 沒有鎖板也沒有柵欄，靠攝影機辨識車牌
　　　　② 離場前到精算機，輸入車格或車牌
　　　　③ 付款完成

提醒　　最不會卡到車，但也因為沒有實體阻擋，
　　　　最容易忘記付款這一步。
```
