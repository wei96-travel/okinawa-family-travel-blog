# Facebook 個人檔案承接計畫

更新：2026-07-24

## 現況

目前簡介：

> 整理沖繩親子自由行｜行程、租車、住宿與帶小孩旅行的實用筆記

問題不是資訊錯，而是像網站分類列。第一次看到的人還不知道「追蹤後能少掉什麼麻煩」，也感覺不到是一個真人在整理。

## 建議主版本

> 第一次帶小孩去沖繩，行程不用排到喘。這裡整理租車、景點、住宿和行李，讓你少查一點、旅途中少慌一點。

這句保留沖繩親子定位，也有明確好處。語氣不裝專家，符合個人帳號的方向。

## 備選版本

較短：

> 第一次帶小孩去沖繩？我把租車、景點、住宿和行李整理好，讓你少查一點、少慌一點。

較有個人感：

> 我把帶小孩去沖繩最容易卡住的事整理在這裡。行程不用排滿，出發前先把真正會用到的看懂。

## 個人檔案四個位置

| 位置 | 要做什麼 | 建議 |
| --- | --- | --- |
| 名稱 | 讓真人感留著 | 暫時保留 `Wei Fan`，不急著改成網站名稱 |
| 簡介 | 說清楚幫誰、解決什麼 | 使用主版本 |
| 連結 | 承接完整攻略 | 放網站首頁；貼文 CTA 寫「完整攻略在個人檔案連結」 |
| 精選／置頂 | 讓新訪客快速認識 | 置頂一篇自我介紹、一篇最實用攻略、一篇表現最好的輪播 |

## 驗證方式

- 簡介更換日記錄為 D0。
- 接下來 14 天不再改簡介。
- 每週記錄個人檔案瀏覽、追蹤增加與網站點擊。
- 若有瀏覽但沒有追蹤，先改置頂內容，不立刻重寫簡介。

Meta 說明頁指出，專業模式可提供內容與受眾洞察，公開內容也有被探索的機會；簡介區的內容是公開資訊。這也是本帳號目前保留專業模式、用個人語氣經營的原因。
