# CTA Library｜60 種 CTA

**禁用**：`完整攻略請點網站`、`詳情請見官網`、`點我看更多`、`手刀收藏`、`快分享給需要的人`、`小編`。

**使用規則**
1. 一則貼文只用一個 CTA。
2. 導流型必須說「過去會看到什麼」，不是說「去看」。
3. `{}` 是變數，發文時換成該篇的具體內容。
4. 同一個 CTA 一個月內不重複使用。
5. 依平台選型：Facebook／Pinterest 用導流型、Instagram 用收藏型、Threads 用互動型、Reels 用收藏或導流型、關注型節制使用。

## F. CTA 檢查

- [ ] 這則只有一個 CTA

- [ ] 導流型有說「過去看到什麼」
