# Social Brand Guide｜社群品牌準則

適用範圍：Facebook、Instagram、Threads、Pinterest、Reels／Shorts／TikTok。
所有平台共用同一套語氣、視覺與 CTA 原則，只有長度和格式不同。

---

## 一、品牌一句話

> 給第一次帶小孩去沖繩的家庭，一份先幫你想過一遍的旅遊筆記。

社群自介統一用這句的變體，不要每個平台各寫一套。

---

## 二、品牌語氣（Tone）

### 五個語氣座標

| 座標 | 我們在哪一端 | 說明 |
| --- | --- | --- |
| 專業 ←→ 親切 | 偏親切 | 用爸媽講話的方式講專業內容 |
| 熱血 ←→ 冷靜 | 偏冷靜 | 不喊「必去」「爆紅」「錯過會後悔」 |
| 華麗 ←→ 樸素 | 偏樸素 | 資訊價值先於形容詞 |
| 權威 ←→ 同行 | 偏同行 | 我們是一起在查資料的家長，不是老師 |
| 完美 ←→ 誠實 | 偏誠實 | 沒去過就說沒去過，會變動就說要查官方 |

### 語氣三原則

1. **先講讀者的處境，再講建議。**
   - ✅「小孩在後座喊想尿尿，前面十公里沒東西。」
   - ❌「沖繩自駕必備知識大公開！」

2. **給判斷，不給清單。**
   - ✅「兩歲以下＋自駕，推車可以不用帶。」
   - ❌「沖繩推車推薦 TOP 10。」

3. **不確定的事講清楚不確定。**
   - ✅「費率各家不同，出發前看一下官網。」
   - ❌「停車費一律 100 円。」

### 用字紅線（NEVER）

- 不用中國用語：不用「攻略滿滿」以外的中國語彙，如 `視頻`、`高清`、`給力`、`性價比`（改用「CP 值」）、`小夥伴`、`打卡地標`（可用「拍照點」）、`民宿老板`、`默認`、`質量`（改「品質」）。
- 不用半形標點與 emoji 亂灑：中文用全形標點；emoji 每則最多 2 個，放句末或段首，不夾在句子中間。
- 不用 AI 味句型：`在這個資訊爆炸的時代`、`讓我們一起`、`不僅…更是…`、`總的來說`、`希望這篇文章能幫到你`。
- 不用誇張詞：`必去`、`神級`、`爆紅`、`零失誤`、`保證`、`最便宜`、`唯一`。
- 不假裝實地體驗：不寫「我住過」「我上次去」「實測」，除非使用者確認真的去過。
- 不寫死會變動的資訊：價格、營業時間、飯店服務一律加「以官方公告為準」。

### 用字綠燈（ALWAYS）

- 稱呼讀者用「你」，自稱用「我們／這邊」。
- 講小孩用具體年齡區間：「兩歲以下」「上小學前」，不要只說「小小孩」。
- 每則貼文至少有一個具體數字或具體地名，避免空泛。
- 出現任何 AI 生成圖，圖說或文案要有「情境示意」四個字。

## 三、品牌個性（Persona）

**設定**：一位已經去過沖繩幾次、習慣把事情先查清楚的家長。個性偏保守、講話直接、不炫耀，但願意把踩過的雷講出來。

| 面向 | 是 | 不是 |
| --- | --- | --- |
| 身分 | 幫你先做功課的同行家長 | 旅遊達人／KOL |
| 態度 | 保守、務實 | 冒險、衝動 |
| 表達 | 短句、具體、有數字 | 抒情、長句、形容詞堆疊 |
| 立場 | 會說「這個不適合你」 | 什麼都推薦 |
| 幽默 | 有一點自嘲式的疲憊感 | 諧星式的搞笑 |

**品牌會說的話**：「這天不要排三個點。」「這件事我也是查了才知道。」「不確定就抓保守一點。」
**品牌不會說的話**：「跟著我這樣玩就對了！」「沖繩我最熟！」「快收藏起來，不然找不到！」

## 四、圖片風格（Visual）

### 色票（與網站完全一致）

| 用途 | 色碼 | 社群用法 |
| --- | --- | --- |
| 主背景 | `#FFFDF8` | 輪播首頁與內頁底色 |
| 次背景 | `#F6EFE4` | 分區塊、表格底、重點框 |
| 邊框 | `#EADFCE` | 卡片描邊、分隔線 |
| 主色 | `#694624` | 標題、CTA 按鈕 |
| 輔色 | `#9A6B43` | 小標、標籤、頁碼 |
| 標題文字 | `#34302B` | 主標 |
| 內文文字 | `#5F594F` | 內文 |
| 提醒色（限用） | `#B4552F` | 「注意」「別踩雷」的重點，一張圖最多一處 |

**禁止**：高飽和藍綠海島風、霓虹漸層、黑底白字、多色彩虹配色。

### 字型

- 中文：思源黑體 / Noto Sans TC（標題 Bold 700，內文 Regular 400）。
- 數字與英文：Inter 或系統無襯線，數字用 Bold 強調。
- 一張圖最多 2 種字重、1 種字型家族。

### 版面規則

- 安全邊界：四邊各留 8%，IG 輪播底部再多留 12% 給介面遮擋。
- 首頁（封面）字級最大，主標最多 16 字、2 行以內。
- 內頁一頁只講一件事，正文不超過 60 字。
- 頁碼固定右下角，用輔色 `#9A6B43`，格式 `3 / 8`。
- 品牌標記固定左下角：小圖示 + `沖繩親子旅遊筆記`，不要每頁換位置。

### 照片與 AI 圖規則

- 目前沒有大量真實照片，以 AI 生成情境圖 + 自製資訊圖為主。詳細 Prompt 見 `ai-image-style-guide.md`。
- AI 圖一律標「情境示意」。
- 不用 AI 圖冒充特定景點、特定飯店、特定店家。要講具體地點時，用資訊圖／地圖式插畫代替。
- 不放可辨識的兒童正臉；人物以背影、側身、局部（手、腳、背包）為主。
- 不放可辨識商標、車牌、店招。

## 五、文字風格（Copy）

### 標題公式（擇一，不要每次都用同一個）

1. **處境 + 問題**：`小孩在車上喊想尿尿，沖繩前面十公里沒東西`
2. **決策題**：`那霸還是恩納？先看你家小孩幾歲`
3. **反常識**：`帶兩歲小孩去沖繩，推車其實可以不帶`
4. **數字整理**：`沖繩投幣停車場有三種，看懂一種其他都一樣`
5. **時間框**：`沖繩最後一天，還車前留 90 分鐘`
6. **恐懼前置**：`租車沒帶到這張，現場可能開不走`

標題不用問號堆疊、不用驚嘆號、不用全大寫。

### 段落節奏

- 第 1 行 = Hook，必須能單獨看懂（很多平台只顯示第一行）。
- 前 3 行內給出結論或最有用的一個資訊，不要留到最後。
- 一段最多 3 行，段與段空一行。
- 條列用「・」或數字，不用 `-` 或 `*`。

### 各平台文案長度

| 平台 | 建議長度 | 說明 |
| --- | --- | --- |
| Facebook | 150–400 字 | 前 2 行是折疊前可見範圍，最關鍵 |
| Instagram | 80–200 字 | 圖是主角，文案補充 + CTA |
| Threads | 60–150 字 | 一則講一件事，可用串接續發 |
| Pinterest | 標題 40 字內 + 描述 100 字內 | 描述要有關鍵字 |
| Reels 字幕 | 每句 8–14 字 | 一屏一句，不換行 |

## 六、CTA 原則

完整 60 種寫法見 `cta-library.md`，這裡只定原則。

1. **一則貼文一個 CTA**，不要同時要人收藏又追蹤又點連結。
2. **CTA 要接續內容，不能通用**。「完整攻略請點網站」這句已被禁用，因為重複且不具體。
3. **CTA 分四種目的**，依平台選：
   - 導流（點連結）：Facebook、Pinterest 主用
   - 互動（留言／提問）：Threads 主用
   - 收藏（存起來）：Instagram 輪播主用
   - 關注（追蹤）：只在內容表現好的貼文用，不每則都放
4. **導流 CTA 要說「過去看到什麼」**，不是說「去看」。
   - ✅「那邊有一張各年齡怎麼選的對照表。」
   - ❌「詳情請見官網。」
5. **不使用**：`點我`、`手刀`、`快收藏不然找不到`、`小編`、`求分享`。

## 七、Hashtag 原則

### 結構：3 層，共 8–15 個

| 層 | 數量 | 作用 | 範例 |
| --- | --- | --- | --- |
| 品牌層 | 1–2 | 累積自有標籤 | `#沖繩親子旅遊筆記` `#OkinawaFamilyNotes` |
| 主題層 | 4–6 | 這則貼文的具體內容 | `#沖繩親子` `#沖繩自駕` `#沖繩住宿` `#沖繩停車` |
| 客群層 | 3–5 | 讀者是誰、在幹嘛 | `#親子旅遊` `#帶小孩出國` `#日本自由行` `#育兒日常` |

### 平台差異

| 平台 | 數量 | 位置 |
| --- | --- | --- |
| Instagram | 10–15 | 放留言第一則，貼文本體保持乾淨 |
| Facebook | 3–5 | 放文末，多了會像機器人 |
| Threads | 1–3 | Threads 標籤機制弱，只放最相關的 |
| Pinterest | 0 | 不用 hashtag，改把關鍵字寫進標題與描述 |
| Reels | 5–8 | 放文案末端 |

### 規則

- 固定的品牌層標籤永遠不變，主題層每則重寫。
- 不用無關的高流量標籤（`#旅遊` `#美食` `#follow4follow`）換曝光。
- 不用超過 20 字的長句標籤。
- 中英文標籤不混寫在同一個標籤裡。
- 每季檢查一次哪些標籤真的帶來觸及，汰換主題層。

## 八、跨平台一致性檢查表

發佈前逐項確認：

- [ ] 用色只用品牌色票，沒有出現藍綠海島風
- [ ] 首頁主標 16 字內、2 行內
- [ ] 有 AI 圖 → 已標「情境示意」
- [ ] 沒有假裝實地體驗的句子
- [ ] 會變動的資訊已加「以官方為準」
- [ ] 沒有中國用語、沒有半形標點
- [ ] 第一行單獨看得懂
- [ ] 只有一個 CTA，且不是「完整攻略請點網站」
- [ ] Hashtag 三層結構齊全，數量符合該平台
- [ ] 對應的網站文章連結正確（可點、無 404）
